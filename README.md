# DokuDay GitHub Pages Legal Site

Esta carpeta contiene los archivos publicos para GitHub Pages.

## Archivos a subir

- `index.html`
- `privacy-policy.html`
- `.nojekyll`

## URL esperada

Si el repo se llama `dokuday-legal` y tu usuario/organizacion de GitHub es `KoratSolutions`, la URL sera:

```txt
https://KoratSolutions.github.io/dokuday-legal/privacy-policy.html
```

Si usas otro usuario o nombre de repo, cambia esos valores en la URL.

## Pasos rapidos

1. Crear un repo publico en GitHub, por ejemplo `dokuday-legal`.
2. Subir estos archivos a la raiz del repo.
3. Ir a Settings > Pages.
4. En Build and deployment seleccionar:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / root
5. Guardar.
6. Esperar 1-5 minutos.
7. Abrir la URL publica y copiarla en Google Play Console.
